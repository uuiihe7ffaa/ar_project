# Aspect Review Analyzer

Простой проект для анализа пользовательских отзывов.
Модель определяет **один главный аспект** отзыва:

* качество
* сервис
* цена
* атмосфера
* доставка

---

## Стек

* Python 3.10+
* scikit-learn
* pandas
* fastapi
* uvicorn
* streamlit

---

# 1. Обучение модели

## 1.1 Подготовка датасета

Формат `training/dataset.csv`:

```
place_type,text,label
кафе,Очень уютное место,атмосфера
ресторан,Стейк пересушен,качество
доставка,Привезли быстро,доставка
```

* `place_type` — тип заведения (кафе, ресторан, доставка)
* `text` — текст отзыва
* `label` — главный аспект

---

## 1.2 Установка зависимостей

```
pip install -r requirements.txt
```

---

## 1.3 Запуск обучения

```
python training/train_model.py
```

После выполнения появятся файлы:

```
models/model.pkl
models/label_encoder.pkl
```

---

## 1.4 Что делает обучение

* преобразует текст в TF-IDF
* обучает классификатор
* сохраняет модель и encoder

---

# 2. Запуск API

## 2.1 Запуск сервера

```
python -m uvicorn app.main:app --reload
```

Сервер поднимется на:

```
http://127.0.0.1:8000
```

---

## 2.2 Пример запроса

POST `/analyze`

```
{
  "text": "Кофе вкусный, но ждать пришлось долго"
}
```

---

## 2.3 Пример ответа

```
{
  "text": "Кофе вкусный, но ждать пришлось долго",
  "aspect": "качество",
  "probabilities": {
    "качество": 0.52,
    "сервис": 0.31,
    "цена": 0.08,
    "атмосфера": 0.05,
    "доставка": 0.04
  }
}
```

# 3. Веб-интерфейс (Streamlit)

## 3.1 Запуск

python -m streamlit run ui/streamlit_app.py

Откроется:

```
http://localhost:8501
```

---

## 3.2 Что есть в интерфейсе

* выбор типа заведения
* ввод текста отзыва

  * главный аспект
  * вероятности по всем аспектам

# 4. Типичные проблемы

## Ошибка: нет колонки labels

Причина: используется старый multi-label код.
Решение: заменить на колонку `label`.

---

## Модель выдает случайные результаты

Причины:

* плохой датасет
* слишком общие тексты
* мало примеров на каждый класс

Решение:

* переписать датасет
* добавить четкие формулировки
* избегать длинных смешанных отзывов

---

## uvicorn не запускается

Используй:

```
python -m uvicorn app.main:app --reload
```
